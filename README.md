## 프로젝트 소개

이 프로젝트는 연인들간의 추억을 공유하는 웹입니다.

✓ 로그인을 통한 추억(게시물) 공유<br>
✓ 게시물(추억)을 쓰기, 읽기, 수정 및 삭제<br>
✓ 해시태그(#), 년,월을 이용한 추억 모아보기<br>
✓ 프로필을 통한 서로의 정보 확인

### 개발기간

- 12/8 ~ 12.29

## 팀 소개

### 😃 FE (Frontend)

- #### 손재학(팀장) : 메인페이지, 캘린더 <br>

- #### 곽민성(팀원) : 글 추가, 상세보기, 수정 및 삭제<br>

- #### 최지은(팀원) : 프로필 및 수정, 파트너 프로필<br>

- #### 김 솔(팀원) : 인트로, 로그인<br>

### 🙂 BE (Backend)

#### 이주영(팀장), 양애영(팀원), 배환(팀원), 우승민(팀원)
